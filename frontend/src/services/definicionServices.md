La carpeta services del frontend contiene funciones encargadas de comunicarse con los endpoints del backend. Sirve para separar la lógica de consumo de API de los componentes React. Los componentes llaman funciones con nombres de negocio, como listarSolicitudes o crearSolicitud, y los services se ocupan de usar Axios, definir endpoint, método HTTP, parámetros y adaptar la respuesta. Esto mejora la organización, reutilización y mantenibilidad del frontend.
